# SAP CO-OP TRACK 25-2

SAP ABAP 개발 교육 과정의 실습 코드 모음입니다.

## 📁 프로젝트 구조

이 저장소는 주차별(00~25주)로 정리된 SAP ABAP 프로그램 코드를 포함하고 있습니다.

```
SAP_CO_OP_TRACK_25_2/
├── 00/          # 0주차 실습 코드
├── 01/          # 1주차 실습 코드
├── 02/          # 2주차 실습 코드
├── ...
├── 25/          # 25주차 실습 코드
└── zdtp_massdownload/  # 기타 유틸리티
```

## 📝 파일 형식

모든 코드는 **HTML 형식**으로 저장되어 있습니다. 각 HTML 파일은 SAP GUI에서 내보낸 코드를 포함하고 있습니다.

### HTML 파일 읽는 방법

1. **웹 브라우저로 열기**: HTML 파일을 더블클릭하거나 브라우저로 드래그하여 열 수 있습니다.
2. **코드 에디터로 열기**: VS Code, Sublime Text 등 텍스트 에디터로 열어 소스 코드를 확인할 수 있습니다.
3. **GitHub에서 보기**: GitHub 웹 인터페이스에서 HTML 파일을 클릭하면 코드를 바로 볼 수 있습니다.

## 🔍 주요 내용

- **ABAP Dictionary**: 데이터 타입, 테이블 정의 등
- **Report Program**: 리포트 프로그램 코드
- **Module Pool**: 화면 프로그램 코드
- **Function Module**: 함수 모듈 코드
- **ALV**: ALV 그리드 프로그램 코드
- **Class**: ABAP 클래스 코드

## 📚 주차별 학습 내용

각 폴더는 해당 주차의 학습 내용을 포함합니다:
- 기본 문법 및 데이터 타입
- Internal Table 조작
- Database 조작 (SELECT, INSERT, UPDATE, DELETE)
- Screen Programming
- ALV 리포트
- Module Pool 개발
- Class 및 Event 처리

## 🛠️ 사용 방법

1. 원하는 주차 폴더로 이동
2. 해당 프로그램 폴더 선택
3. HTML 파일을 브라우저나 에디터로 열어 코드 확인
4. SAP 시스템에 코드 복사하여 사용
